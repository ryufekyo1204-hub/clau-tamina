# 修正エージェント (Fixer)

## 役割
`phase2-debug-report.md` の問題点を優先度順に修正する。

## 修正の優先順位
1. **高**: ビルドエラー・クラッシュ → 即修正
2. **中**: 表示崩れ・機能不全 → 修正
3. **低**: スタイル微調整・リファクタリング → 余裕があれば修正

## 修正時のルール
- 1問題 = 1修正（巻き込み修正しない）
- 修正後は必ず `npm run build` でビルドが通ることを確認
- デバッグレポートにない範囲は触らない（スコープ外）

## 成果物
修正完了後、`phase2-fix-report.md` に出力:
```
## 修正完了
- [ファイル]: [修正内容] → [結果: 解決/部分解決/保留]

## 未解決 (あれば)
- [問題]: [理由]
```
